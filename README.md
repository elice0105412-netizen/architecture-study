# Software Architecture Study

본 저장소는 소프트웨어 아키텍트 진로를 탐색하며  
시스템 구조 및 설계 개념을 학습하고 정리하기 위한 개인 학습 기록 공간입니다.

단순한 기능 구현을 넘어,  
소프트웨어가 어떤 구조로 설계되고 각 구성 요소가 어떻게 연결되는지를 이해하는 것을 목표로 합니다.

---

## 학습 목적
- 소프트웨어 구조 및 아키텍처 기초 개념 이해
- 설계 관점에서 시스템과 코드를 바라보는 사고력 향상
- GitHub를 활용한 학습 기록 관리 및 자기주도 학습 습관 형성

---

## 주요 학습 내용
- 소프트웨어 구조 설계 기초
- 레이어드 아키텍처 개념
- 모듈화 및 역할 분리의 중요성
- GitHub 기반 학습 기록 관리

---

## 학습 방식
- 국내 온라인 학습 플랫폼을 활용한 이론 학습
- 학습 내용 정리 및 기록
- 간단한 구조 설계 예제 작성
- 커밋 기록을 통한 학습 과정 관리

---

## 진행 현황
- 2025.03 : 구조 설계 기초 개념 학습 시작
- 학습 내용 정리 및 기록 진행 중
- 추후 설계 예제 및 학습 노트 지속 추가 예정

본 저장소는 학습 및 진로 탐색 목적의 개인 기록용 저장소입니다.
